# Лабораторная работа по основам алгоритмизации и программирования. Сравнительный анализ алгоритмов сортировки

Данный лабораторный проект предназначен для измерения и сравнения процессорного времени работы различных алгоритмов сортировки на массивах разного размера и с разным типом заполнения данных.

## 🚀 Возможности
* **Сравнение 5 алгоритмов**: Bubble Sort, Selection Sort, Quick Sort, Merge Sort, Heap Sort.
* **4 типа генерации данных**: 
  * случайные числа
  * отсортированный массив
  * массив, отсортированный в обратном порядке
  * массив с большим количеством дубликатов
* **Масштабируемость**: Тесты запускаются на массивах размером от 100 до 50 000 элементов.
* **Визуализация**: Вывод результатов в терминал полностью раскрашен ANSI-цветами и выровнен в удобную таблицу для анализа.

## 📁 Структура проекта

```text
├── generators/       # Генераторы тестовых массивов
│   ├── generateArray.c
│   └── generateArray.h
├── algorithms/       # Реализация алгоритмов сортировки
│   ├── bubbleSort.c
│   ├── heapSort.c
│   ├── mergeSort.c
│   ├── quickSort.c
│   ├── selectSort.c
│   └── sort.h
├── obj/              # Автоматически создаваемая папка для объектных файлов (.o)
├── main.c            # Точка входа
├── Makefile          # Скрипт автоматической сборки
└── sorting           # Итоговый исполняемый файл (появится после сборки)
```

## 🛠️ Сборка и запуск

### 1. Компиляция 
```bash
make
```

### 2. Запуск программы
```bash
./sorting
```

### 3. Полная очистка
```bash
make clean
```
